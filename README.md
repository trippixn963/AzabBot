# AzabBot

![Banner](images/BANNER.gif)

Personal Discord bot for discord.gg/syria

## Setup

```bash
pip install -r requirements.txt
python main.py
```

## Config

Create `.env` file:
```
DISCORD_TOKEN=your_token
OPENAI_API_KEY=your_key
DEVELOPER_ID=your_id
TARGET_ROLE_ID=muted_role_id
PRISON_CHANNEL_IDS=channel_id
```
