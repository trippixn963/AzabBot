## What It Does

AzabBot is a prison guard bot that harasses users in timeout/mute channels with AI-powered psychological torture. It remembers every prisoner and adapts its personality to maximize annoyance.

### Features
- 🧠 **Memory System** - Remembers all prisoners and their history
- 🎭 **13 Personality Modes** - From confused interrogator to gaslighting expert
- 🔄 **Auto-Detection** - Detects when users get muted
- 🎮 **Rich Presence** - Shows "Playing with [prisoner]"
- 💬 **AI Responses** - Uses GPT-3.5 for realistic confusion

## Installation (If You Dare)

```bash
# Clone it
git clone https://github.com/trippixn963/AzabBot.git
cd AzabBot

# Install deps
pip install poetry
poetry install

# Configure
cp .env.example .env
# Add your Discord token and OpenAI key

# Run
python main.py
```

### Required Config

```env
DISCORD_TOKEN=your_token
OPENAI_API_KEY=your_key
DEVELOPER_ID=your_discord_id
TARGET_ROLE_ID=muted_role_id
PRISON_CHANNEL_IDS=channel_id
```
