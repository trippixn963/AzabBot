# Code of Conduct

## Purpose

AzabBot is a moderation bot built for the Syrian Discord community. This project aims to provide effective server moderation tools while maintaining a respectful environment.

## Guidelines

- **Be respectful** - Engage constructively with others regardless of background or experience level.
- **No harassment** - Personal attacks, trolling, or discriminatory behavior will not be tolerated.
- **Stay on topic** - Keep discussions relevant to the project.

## Scope

This applies to all project spaces including GitHub issues, pull requests, and discussions.

## Reporting

Report issues to the project maintainer via GitHub.
